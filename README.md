# Phonebook CI/CD

This repository contains a simple Phonebook application with a CI/CD pipeline built using GitHub Actions and deployed to Fly.io.

---

## 🚀 Deployed Application

👉 https://phonebook-cicd-restless-sea-2695.fly.dev

---

## 📦 Features

- Express backend
- Dockerized application
- CI/CD pipeline with GitHub Actions
- Automatic deployment to Fly.io
- Discord notifications for success/failure
- Periodic health check

---

## ⚙️ CI/CD Pipeline

The pipeline includes:

- Install dependencies
- Build application
- Run tests
- Deploy to Fly.io
- Send Discord notifications
- Tag releases
- Scheduled health check
